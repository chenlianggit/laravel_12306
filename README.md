# laravel_12306
微信小程序 12306抢票系统
#系统组成

| 程序 | 版本 | 
| --- | --- |
| Linux |  Centos7.5 | 
| Mysql | 5.6 | 
|  PHP| 7.2| 
|  Python| 3.6 | 
|微信小程序|官方|
 
# 整体流程
1. 在微信小程序获取列车表
2. 登陆12306账号->获取联系人->选择需要订票的人(多选)
3. 将选择的车次、座位、联系人...信息POST到PHP
4. php做校验和关联储存，将该条主键POST到Python抢票接口
5. Python根据ID查询mysql,进行Python流程


# Python流程

1. 用户登陆
1. 正在初始化订票页面...
2. 查询余票...
3. 正在确认用户登录状态...
4. 正在提交车票预定信息...

8. 正在确认配置信息...
9. 排队等待中获取Orderid...
10. 抢票成功!
